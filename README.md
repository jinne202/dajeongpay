# :sparkler: Dejeongpay   

```sh
npm run dev
```

해당 프로젝트는 친구의 졸업전시 배포용 사이트입니다.   
디자이너의 요구로 1920 해상도에만 원활하게 보이게 제작되었습니다.   

https://dajeong-pay.netlify.app


***

- React / Next.js / React-Redux / Styled Component

***

##### 기능설명

- redux를 사용한 카운터
- 라디오 박스
- %에 맞게 css 조절
- netlify 배포
